# LeetTrack 🧑‍💻

Track your true LeetCode progress across multiple accounts.

## The Problem
Many developers have multiple LeetCode accounts but no way to know their 
true unique problem count. You might think you've solved 170 problems across 
2 accounts, but some could be duplicates!

## What LeetTrack does
- Add up to 5 LeetCode usernames
- See Easy / Medium / Hard breakdown per account
- View combined stats across all accounts
- Connect accounts securely using session cookies
- Find your TRUE unique problem count
- See exactly which problems you solved on multiple accounts

## Tech Stack
- Next.js 15 (App Router)
- Tailwind CSS
- LeetCode GraphQL API
- Recharts

## Getting Started
npm install
npm run dev

## Privacy
Session cookies are never stored on our servers. 
All processing happens in real time and cookies are discarded immediately.
